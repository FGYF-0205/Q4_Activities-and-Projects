JavaScript Arithmetic Operators

Did you know? That mathematics is also involved in computer programming. In today's topic, we will be solving basic operations using the available operators in JavaScript.

GIVEN:

a = 10, b = 5;

  1. a + b = 15
  2. b - a = -5
  3. a * b = 50
  4. b / a = 1/2
  5. a % 2 = 0
